## 项目描述
   该demo是一套整套权限管理系统，通过后端配置，可以实现用户的菜单权限/按钮权限/接口权限的控制。
   
## 技术栈
#### 后端 node + koa2   
#### 前端
**vue版本:** vue + vuex + router +axios + element UI

**react版本:** react + antd + (dva)
